                    Mohammad Ashour Jestin Abraham • 9 months ago

                    Hey Jestin Abraham, thank you. Well Flutter's i18n library is based on the intl Dart library, which does seem to support a select format for conditional logic. I don't think everything in intl is available in Flutter's lib, so I'm not sure if it's easy or straightforward to use the select format in your ARB files and have that auto-generate.

                    Of course you can go with the intl package directly an bypass Flutter's i18n lib. We have a somewhat older but should still be largely relevant article about doing that.

                    And you might just want to deal with the (hopefully few) instances where you want conditional logic in your messages through a helper class that wraps more static messages. That way you can just have your conditional logic in plain old Dart.
